- ✅ Game reset functionality
- ✅ Interactive UI with smooth user experience
- ✅ Responsive design (Mobile + Tablet + Desktop compatible)
- ✅ Clean and structured code

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the game
- **CSS3** – Styling and responsive layout
- **JavaScript (Vanilla JS)** – Game logic and interactivity

---

## 🧠 Game Logic

- The game is played on a 3×3 grid.
- Players take turns marking a cell (X or O).
- After every move, the script checks:
  - Winning combinations
  - Draw condition
- When a player wins or the game draws, further moves are disabled.
- Reset button allows restarting the game instantly.
